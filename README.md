# Tax-calculator

Weekend Hack With [Electron](https://www.electron.com/).

# Project

Project built is a tax calculator that helps you calculate your monthly tax and ssnit
contributions. Calculation is based on values from [GRA's Official Website](http://www.gra.gov.gh/index.php/tax-information/income-tax)

# How To Use

`git clone https://github.com/ankorGH/tax-calculator.git`
`npm install`
`npm start`
